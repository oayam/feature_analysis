    "navigationId": "demo_navigation_name",

```
{
    "name": "demo_name_primary",
    "mode": 1,
    "navigationId": "demo_navigation_name",
    "param":
    {
        "desc": "demo_param_desc"	
    },
    "launchMode": 1
},
```

被做了launchMode标注的页面，在恢复时会使用对应的launchMode进行页面加载。具体来说，调用`CallPushDestinationInner`函数时需要将json中的launchMode考虑在内
